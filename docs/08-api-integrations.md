# API Integrations — AI Sales OS

## Архитектура интеграций

Все внешние API инкапсулированы в **Integration Adapters** — модульные классы с единым интерфейсом. Это позволяет заменять провайдера (например, Mailgun → Brevo) без изменений в бизнес-логике.

```typescript
interface IntegrationAdapter {
  name: string;
  test(): Promise<{ ok: boolean; error?: string }>;
  // Специфичные методы в наследниках
}
```

API-ключи хранятся в таблице `api_keys` (зашифровано AES-256). Workspace хранит свои ключи или использует системные дефолтные.

## 1. Источники лидов

### 2ГИС API

| Параметр | Значение |
|---------|---------|
| Официальный? | ✅ Да |
| Документация | https://docs.2gis.com |
| Бесплатный tier | Demo key: 1,000 req/день; ограниченные endpoints |
| Платный | От ₽5,000/мес; неограниченный поиск по каталогу |
| Лицензия использования | Нельзя перепродавать сырые данные; отображение карт требует attribution |

**Используемые endpoints:**
```
GET /3.0/items/geocode         → геокодирование адресов
GET /3.0/items                 → поиск организаций по категории и городу
GET /3.0/items/{id}            → детальная информация об организации
```

**Что получаем:** название, адрес, телефоны, категория, часы работы, рейтинг, фото.

**Чего нет:** email руководителя, ИНН, финансовые данные.

**Стратегия использования:**
1. Поиск по `rubric_id` (отрасль) + `city_id` → получаем список
2. Обогащаем ИНН через ЕГРЮЛ (matching по названию + адресу)
3. Доменный email ищем через Hunter.io

### HH.ru API (api.hh.ru)

| Параметр | Значение |
|---------|---------|
| Официальный? | ✅ Да |
| Документация | https://api.hh.ru/openapi/redoc |
| Бесплатный tier | OAuth2 доступ бесплатен для просмотра публичных данных |
| Ограничения | Персональные данные соискателей — только для работодателей с договором |

**Используемые endpoints:**
```
GET /employers?text=&area=&industry=    → поиск работодателей
GET /employers/{id}                      → детали работодателя
GET /vacancies?employer_id=              → вакансии работодателя
```

**Что получаем:**
- **Работодатель**: название, сайт, отрасль, город, количество сотрудников, открытые вакансии
- **Вакансии**: должности, навыки, зарплата, описание (=боли компании, стек технологий)

**Сигналы из вакансий:**
- `"Ищем руководителя транспортного отдела"` → рост логистической функции
- `"Требуется водитель-экспедитор x5"` → масштабирование автопарка
- Зарплатная вилка → косвенный размер компании

### ЕГРЮЛ (egrul.nalog.ru)

| Параметр | Значение |
|---------|---------|
| Официальный? | ✅ Да (ФНС РФ) |
| Доступ | Открытые данные, бесплатно |
| API | Неофициальный (парсинг HTML или через агрегаторов) |
| Альтернативы | Dadata.ru API (платно, ₽0.50/запрос), СБИС API |

**Что получаем:** официальное наименование, ИНН, ОГРН, ОКВЭД, статус (действующее/ликвидировано), ФИО руководителя, дата регистрации, уставный капитал.

**Стратегия:** используем Dadata.ru API как агрегатор (быстро, структурировано, кэширует). При высоком объёме — прямой парсинг ЕГРЮЛ с уважительными задержками.

### VK API

| Параметр | Значение |
|---------|---------|
| Официальный? | ✅ Да |
| Документация | https://dev.vk.com/api/access-token/authcode |
| Бесплатный tier | 3 req/sec для Client Credentials; Groups API бесплатен |
| Ограничения | Личные сообщения нельзя; только публичные группы и страницы |

**Используемые методы:**
```
groups.search      → поиск групп по ключевым словам и типу "page"
groups.getById     → детали группы (сайт, контакты, описание)
wall.get           → посты (сигналы активности компании)
```

**Применение:** дополнительный источник сигналов о компании; иногда в описании группы есть контактный email.

## 2. Обогащение данных

### Hunter.io

| Параметр | Значение |
|---------|---------|
| Тип | Email finder по домену |
| Бесплатный tier | 25 поисков/мес |
| Платный | $49/мес (500 поисков), $99/мес (2,000 поисков) |

**Endpoints:**
```
GET /domain-search?domain=company.ru    → все найденные email на домене
GET /email-finder?domain=&first_name=&last_name=   → email конкретного человека
GET /email-verifier?email=              → верификация
```

**Место в waterfall:** первый провайдер при поиске email.

---

### Snov.io

| Параметр | Значение |
|---------|---------|
| Тип | Email finder + верификация |
| Бесплатный tier | 50 кредитов/мес |
| Платный | $39/мес (1,000 кредитов) |

**Место в waterfall:** второй провайдер если Hunter не нашёл.

---

### Apollo.io

| Параметр | Значение |
|---------|---------|
| Тип | B2B база данных + email/phone |
| Бесплатный tier | 50 email export/мес |
| Платный | $49/мес (1,000 credits) |

**Место в waterfall:** третий провайдер; лучше для международных компаний.

---

### Dadata.ru

| Параметр | Значение |
|---------|---------|
| Тип | Агрегатор данных ФНС, ФИАС, СБИС (РФ-специфика) |
| Бесплатный tier | 10,000 запросов/сутки |
| Платный | ₽0.50–2.00 за запрос при превышении |

**Endpoints:**
```
POST /suggest/party     → поиск компании по ИНН или названию
POST /findById/party    → детали по ИНН
POST /suggest/address   → стандартизация адреса
```

**Незаменим для РФ-рынка**: official ФИО руководителя, статус компании, ОКВЭД.

## 3. Email отправка

### Mailgun

| Параметр | Значение |
|---------|---------|
| Бесплатный tier | 100 emails/день (3 месяца) |
| Платный | $35/мес (50,000 писем) |
| Особенности | Лучший deliverability в industry; инлайн трекинг; webhooks |

**Используемые API:**
```
POST /v3/{domain}/messages      → отправка
GET  /v3/{domain}/events        → события (открытия, клики)
POST /v3/routes                 → webhooks для tracking
```

**Webhooks получаем:**
- `delivered`, `opened`, `clicked`, `bounced`, `complained`, `unsubscribed`

**Резервный провайдер.** Аналогичный функционал, €25/мес за 20,000 писем. Хорошая доставляемость в РФ.

## 5. Мессенджеры

### Telegram Bot API

| Параметр | Значение |
|---------|---------|
| Официальный? | ✅ Да |
| Стоимость | Бесплатно |
| Rate limits | 30 сообщений/сек на бота; 1 сообщение/сек конкретному пользователю |
| Ограничения ToS | Только opt-in пользователи; запрещён спам |

**Использование в системе:**
1. **Уведомления SDR**: "Лид [Company X] ответил — хочет звонок" (inline кнопки: Позвонить / Отложить)
2. **Команды**: `/stats` — статистика кампании, `/pause campaign_id` — пауза
3. **Оперативный мониторинг**: алерты при падении системы

**НЕ используем** для cold outreach — это нарушение ToS Telegram.

## 7. Резервные и альтернативные подходы

| Сервис | Статус | Наш подход |
|-------|--------|-----------|
| Avito | Нет публичного API для чтения | Используем официальный Avito Business API только для авторизованных рекламодателей |
| LinkedIn | API очень ограничен | LinkedIn Sales Navigator через официальный партнёрский API ($) или ручной экспорт |
| 2GIS scraping | Нарушение ToS | Только официальный платный API |
| Yell/Zoon | Нет API | Не используем; 2ГИС покрывает ту же задачу лучше |
