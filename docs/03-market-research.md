## 2. Рынок Route Optimization в России

### Общая картина
- Рынок TMS (Transport Management Systems) в России: ~18 млрд руб. (2024), рост 12–15%/год
- После ухода SAP/Oracle/JDA из РФ образовался вакуум в mid-market сегменте
- Ключевые игроки: Яндекс Маршрутизация, Maxoptra, РеутERP, 1С:TMS, EasyWay
- Основные боли клиентов: ручное планирование, невозможность пересчёта при изменениях, нет API для интеграции с WMS

### Целевой сегмент для MVP
**Компании с автопарком 20–500 машин:**
- Дистрибьюторы продуктов питания, FMCG
- Курьерские службы (региональные)
- Строительные компании с доставкой материалов
- Аптечные сети с собственной доставкой
- Ритейл с последней милей

### Количество потенциальных клиентов (РФ)
- По данным ФНС и 2ГИС: ~45,000 компаний с кодами ОКВЭД 49.41, 52.29, 52.10
- Из них с автопарком 20+ машин — ~8,000–12,000
- Адресный рынок (TAM): 8,000 × $3,600/год = ~$29M

### Конкуренты первого продукта
| Конкурент | Цена/мес | Слабости |
|-----------|---------|---------|
| Яндекс Маршрутизация | от ₽15,000 | Нет CRM, нет API-продаж, Enterprise-ориентация |
| Maxoptra | $300–800 | Английский интерфейс, медленная поддержка в РФ |
| 1С:TMS | от ₽50,000 + внедрение | Долгое внедрение, устаревший UX |
| EasyWay | ₽10,000–30,000 | Ограниченная аналитика |
| Самописные решения | ₽0 (но разработка) | Не масштабируется, нет поддержки |

**Наша позиция:** быстрое внедрение (SaaS, без интегратора) + AI-персонализация продаж + прозрачный ROI.

## 4. Источники данных (РФ-специфика)

### Официальные API
| Источник | Что даёт | Бесплатный лимит | Ограничения |
|---------|---------|-----------------|------------|
| **2ГИС API** | Компании по категориям, адреса, телефоны | Demo key (1000 req/день), платный от ₽5,000/мес | Нет email, нет ФИО руководителя |
| **HH.ru API** | Компании, вакансии (косвенно: боли, стек, рост) | OAuth, бесплатно для просмотра | Персональные данные защищены, нет прямых email |
| **Yandex Maps API** | Геокодирование, POI, маршруты | 2.5M запросов/год бесплатно | Нет контактов компаний |
| **Yandex Direct API** | Рекламные данные конкурентов | Требует аккаунт рекламодателя | Не для парсинга лидов |
| **Telegram Bot API** | Рассылка, бот-ответы | Бесплатно, 30 msg/sec | ToS: не спам, только opt-in |
| **VK API** | Группы компаний, реклама | Бесплатно (Groups API) | Rate limit: 3 req/sec, нельзя парсить личные сообщения |
| **ЕГРЮЛ (ФНС)** | Официальные данные компаний, ИНН, директор | Бесплатно через egrul.nalog.ru | Нет email/телефонов, только официальные данные |

### Полулегальные / Альтернативные источники
| Источник | Подход | Риски |
|---------|--------|-------|
| **Avito** | Официального API нет. Avito Business API (beta) — для размещающих объявления. Парсинг — нарушение ToS | Бан аккаунта, IP, юридические претензии |
| **LinkedIn** | Официальный API очень ограничен. LinkedIn Sales Navigator — $99+/мес | Парсинг — прямое нарушение ToS, судебные прецеденты (hiQ v. LinkedIn) |
| **Zoon, Yell, Flamp** | Парсинг — нарушение ToS | Риски аналогичны |

### Рекомендованный подход (легальный)
1. **2ГИС + ЕГРЮЛ** — основа: юрлицо, адрес, категория
2. **HH.ru** — косвенное обогащение через вакансии (боли, стек, рост)
3. **VK API** — группы и публичные страницы компаний
4. **Telegram** — публичные каналы и группы (через MTProto, только публичные)
5. **Hunter.io / Snov.io** — поиск email по домену (платно, но легально)
6. **Clearbit / Apollo** — обогащение для международных компаний

## 5. Технологические тренды

### AI в продажах (2024–2026)
- **LLM-персонализация** вытесняет шаблонный маркетинг: reply rate растёт с 2–3% до 8–15% при правильной персонализации
- **AI SDR** (Artisan, 11x.ai, Ava) — новая категория; оценка рынка $500M к 2026
- **Voice AI** (ElevenLabs, Vapi) — автоматические холодные звонки, готово для production
- **Browser Use** / **Computer Use** — агенты, управляющие браузером, убирают потребность в API
- **RAG + Company Knowledge** — агент знает о компании больше, чем её сайт

### Российская специфика
- После 2022 года уход западных CRM создал спрос на отечественные решения
- Импортозамещение: госкомпании обязаны использовать российский ПО — потенциальный рынок
- Мессенджер-ориентированная культура: Telegram > Email для деловой переписки в ряде сегментов
