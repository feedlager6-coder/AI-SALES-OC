# Product Goals — AI Sales OS

## Стратегические цели (12–18 месяцев)

### G1. Запустить работающий MVP за 90 дней
- Полный цикл: поиск лидов → обогащение → персонализированный аутрич → трекинг → CRM
- Первый платящий клиент в первые 30 дней после запуска

### G2. Доказать product-market fit на транспортной вертикали
- 10 платящих клиентов с положительным NPS (> 40)
- Измеримое сокращение времени цикла продаж у клиентов

### G3. Создать переиспользуемое ядро платформы
- Новую вертикаль можно подключить за 2 недели (меняется только ICP + промпты + источники данных)
- Все компоненты задокументированы и покрыты тестами

### G4. Достичь операционной масштабируемости
- Система должна обрабатывать 10,000 лидов/день без деградации
- Один оператор может управлять 5 параллельными кампаниями

## Продуктовые цели по подсистемам

### CRM
- Контакт/компания создаются автоматически из любого источника (email, парсинг, импорт)
- Полная история взаимодействий на каждой карточке
- Кастомные поля без разработки (low-code конфигурация)
- Пайплайн с автоматическим движением сделки по триггерам

### Lead Generation
- Поиск лидов по 5+ источникам (2ГИС, HH.ru, ОФД/ЕГРЮЛ, Avito, отраслевые справочники)
- Дедупликация по ИНН, домену, email
- ICP-скоринг каждого лида (0–100)

### Lead Enrichment
- Водопадное обогащение: если источник A не вернул email — пробуем B, потом C
- Цель: email (verify rate > 85%), телефон, LinkedIn, ФИО ЛПР, выручка, численность

### Email Outreach
- Мультишаговые последовательности (up to 8 touchpoints)
- A/B тесты темы, тела, CTA
- Автоматическое прогревание отправляющих ящиков
- Отслеживание: open, click, reply, bounce, unsubscribe

### AI Layer
- Генерация персонализированных писем (не шаблон, а реальный контекст компании)
- Классификация ответов: заинтересован / не заинтересован / просьба перезвонить / юридический запрет
- Рекомендации следующего шага для SDR
- Отчёты: почему лид не конвертировался

### Analytics
- Воронка на каждом этапе (лид → квалификация → встреча → сделка)
- Revenue attribution: какой канал, сообщение, время принесли деньги
- Дашборд для руководителя (топ-5 KPI в реальном времени)

## Антицели (что мы НЕ строим)

| Антицель | Почему |
|----------|--------|
| Полноценная ERP | Не наша задача; интегрируемся с 1С/Битрикс |
| Собственная email-инфраструктура | Берём готовые (Mailgun/Brevo/SES) |
| Мобильное приложение в MVP | Веб-first, mobile — v2 |
| B2C функционал | Только B2B, другая логика и compliance |
| Замена людей-продавцов | AI — усилитель, не замена |

## KPI по кварталам

| Квартал | KPI |
|---------|-----|
| Q1 (MVP) | 1 вертикаль, 3 активные кампании, 1,000 лидов/мес обрабатывается |
| Q2 | 10 платящих клиентов, email open rate > 35%, reply rate > 8% |
| Q3 | 2-я вертикаль, MRR $20K, < 3% churn |
| Q4 | MRR $50K, CAC payback < 6 мес, NPS > 45 |

## Пользователи и их цели

| Роль | Цель в системе |
|------|---------------|
| **Founder / CEO** | Видеть воронку и выручку в реальном времени |
| **SDR / Sales Manager** | Получать готовые задачи, не искать лидов вручную |
| **Head of Sales** | Управлять кампаниями, настраивать ICP, анализировать эффективность |
| **Developer (нас)** | Добавлять новые вертикали через конфиг, а не код |
